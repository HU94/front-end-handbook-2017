# 学习网站性能优化

> 页面性能优化，又称 WPO，亦或是站点优化，它是一门提升页面在用户 Web 浏览器中下载及展现速度的学问。随着全球平均网速的不断提升，对于网站管理员及站长们来说，关注用户所访问网站的渲染时间是很有必要的。
