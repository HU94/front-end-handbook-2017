# 自学

本节重点介绍适合前端开发人员自我指导学习和自我指导职业发展的免费及付费资源（视频课程、图书等）。

以下所提及的资料包括免费和付费资料，其中付费资料将使用 [$] 标注。

作者相信，任何有决心且愿意付出的人都能够通过自我学习成为一名前端开发者。你只需要一台连接到互联网的计算机，以及一些用来购买书籍和视频课程的经费。
